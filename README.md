# AI Student Support Chatbot for myUMBC Portal

This is a small AI-powered chatbot designed to assist students in navigating the myUMBC portal. The chatbot can retrieve information from a FAQ dataset using semantic similarity, providing quick and accurate answers to common queries.

The project is built with **Flask**, **FAISS**, and **Sentence Transformers**.

---

## Features

- Quick access to frequently asked questions (FAQs)
- Natural Language Processing (NLP) for understanding user queries
- Semantic similarity search for accurate responses
- URL retrieval and direct communication
- UMBC-themed user interface

---

## Technology Stack

- **Python 3**
- **Flask** – Web framework for hosting the chatbot interface
- **FAISS** – Efficient similarity search for vector embeddings
- **Sentence Transformers** – Converts text queries and FAQs into vector embeddings
- **Visual Studio Code** – Recommended IDE for running the project
